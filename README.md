# 여수 365 섬-순환(Island-Loop) 자립 모델 분석 프로젝트

본 프로젝트는 **2026 여수 세계섬박람회**를 대비하여 여수시 유인도의 정주 여건을 개선하고, 지속 가능한 섬 관리를 위한 **'자원 순환 및 에너지 자립 모델'**을 제안하기 위해 기획되었습니다.

## 1. 분석 배경 및 목적
- **문제 정의**: 섬 지역의 높은 폐기물 처리 비용(육지 운송)과 에너지 공급의 외부 의존성 해결 필요.
- **분석 목적**: 공공데이터를 활용해 섬별 쓰레기 발생량과 에너지 자립 잠재력을 분석하고, 최우선적으로 스마트 시설 도입이 필요한 '거점 섬'을 선정함.

## 2. 사용 데이터 (`row_data` 폴더)
- **전라남도 여수시_유인도서현황**: 섬별 인구, 면적, 가구수 등 기본 지표
- **전라남도 여수시_관광객현황**: 연도별/분기별 여수시 방문객 추이
- **전라남도 여수시_연도별 음식물 쓰레기 처리비용**: 경제성 분석을 위한 시계열 비용 데이터
- **전라남도 여수시_시민여객선 운임지원_여객선 항로 정보**: 섬별 물류 접근성 파악용

## 3. 주요 분석 내용
1. **관광객 기반 쓰레기 발생량 예측**: 섬별 면적과 인구 비중을 고려해 실질적인 일일 쓰레기 배출량 추정.
2. **에너지 자립도 시뮬레이션**: 섬별 가용 면적 대비 태양광 발전 잠재량과 에너지 수요량 비교.
3. **우선순위 거점 도출**: 쓰레기 발생량(50%), 에너지 미자립도(30%), 배편 유무(20%)를 결합한 복합 점수 산출.

## 4. 실행 방법
```bash
# 필요한 라이브러리: pandas, numpy, matplotlib, seaborn
python analysis.py
```

## 5. 결과물 (`final_result` 폴더)
- **분석_결과_차트.png**: 쓰레기량 대비 에너지 자립도를 시각화한 우선순위 매트릭스.
- **아이디어_기획_메모.txt**: 공모전 제안서 작성 시 활용할 경제적 수치 및 추천 섬 목록.
- **최종_데이터_정리.csv**: 모든 분석 지표가 통합된 로데이터.
